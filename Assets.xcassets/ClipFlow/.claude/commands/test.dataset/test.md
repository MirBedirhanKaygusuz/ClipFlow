Belirtilen modül/fonksiyon için test yaz.

1. Mevcut kodu oku ve anla.
2. pytest kullan.
3. Şu test türlerini yaz:
   - Happy path (normal çalışma)
   - Edge case (boş input, çok büyük dosya, geçersiz format)
   - Error case (exception fırlatması gereken durumlar)
4. Mock kullan: FFmpeg subprocess, R2 upload, Whisper API.
5. Test dosya adı: test_{modül_adı}.py
6. Her test fonksiyonuna açıklayıcı docstring yaz.

Hedef: $ARGUMENTS
